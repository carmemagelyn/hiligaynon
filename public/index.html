SPEAK Hiligaynon

Interactive language learning for Korean and Chinese students at Hua Siong College of Iloilo

🌍

Multi-Language Support

Choose between Korean (한국어) or Chinese (中文) as your native language for personalized learning.

🎯

Interactive Learning

Flashcards, quizzes, pronunciation practice, and real-world examples for effective learning.

📚

Practical Vocabulary

Learn essential Hiligaynon words for daily living situations in Iloilo.

📊

Progress Tracking

Monitor your learning progress with detailed analytics and achievement system.

🚀 Quick Setup

1. Install Dependencies

npm install

2. Start Development Server

npm run dev

3. Open Browser

http://localhost:3000
